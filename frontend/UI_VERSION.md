# Версия UI (май 2026)

Сохранённая ветка интерфейса с обновлённым дизайном:

- Шрифт DM Sans, зелёная палитра, карточки и вкладки
- Русские подписи: роли, тип снаряжения, статус похода, тип назначения, пол
- Графики (Recharts): нагрузка участников, состав раскладки, снаряжение по типу, вес продуктов

Установка графиков:

```bash
cd frontend
npm install
npm run dev
```
