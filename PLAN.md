## API (кратко)

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Вход (cookie сессии) |
| GET | `/api/auth/me` | Текущий пользователь |
| GET | `/api/reference/suggested-weight?gender=FEMALE` | Реком. вес |
| CRUD | `/api/hikes`, `/participants`, `/gear`, `/food` | Поход |
| POST | `/api/hikes/{id}/load-plan/generate` | Раскладка |
| GET | `/api/hikes/{id}/load-plan/export` | Экспорт JSON |
| GET | `/api/hikes/{id}/load-plan/my` | Моя раскладка |

**Демо-вход:** `organizer@hike.local` / `demo`
